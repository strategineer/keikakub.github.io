Battleships

  • Read Code
  • cpp

Networked 2D top down turn-based strategy game inspired by the Battleships board game.

Battleships was developed using C++ and SDL in a team of three programmers composed of Benjamin Lewis, Franz-Philippe Garcia and myself in my second year at McGill University within the context of the Software Engineering course led by Jörg Kienzle.

I worked specifically on the game client, the graphics code and the art.
